Курсовая работа

Курсовая работа посвящена теме "Разработка веб-приложения с использованием современных технологий".

Основная цель работы — создать функциональное веб-приложение с адаптивным дизайном, системой аутентификации и взаимодействием с сервером.

Основные разделы курсовой работы

  • Введение
  • Технологический стек (HTML, CSS, JS, React, Node.js)
  • Аутентификация и авторизация
  • Работа с базой данных
  • Деплой приложения на сервер
  • Заключение
